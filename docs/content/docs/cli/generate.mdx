---
title: generate command
description: Generate any profile asset — banner, QR, word cloud, art, skills, or README.
---

## Synopsis

```bash
uv run python -m scripts.cli generate ENTITY_TYPE [OPTIONS]
```

`ENTITY_TYPE` is case-insensitive.

## Common options

| Option | Applies to | Notes |
|---|---|---|
| `--config-path` | Most | Path to `config.yaml` (default: `~/.w4w-config.yaml`) |
| `--output-path` | banner, word_cloud | Override config output path |

## `generate banner`

Generates a light-mode SVG banner and automatically derives a dark-mode variant.

```bash
uv run python -m scripts.cli generate banner \
  --config-path config.yaml \
  --output-path .github/assets/img/banner.svg \
  --title "My Name" \
  --subtitle "Developer" \
  --width 1200 \
  --height 630
```

| Option | Notes |
|---|---|
| `--title` | Overrides `banner_settings.title` |
| `--subtitle` | Overrides `banner_settings.subtitle` |
| `--width` | Overrides `banner_settings.width` |
| `--height` | Overrides `banner_settings.height` |
| `--optimize/--no-optimize` | Toggle SVGO post-processing |

CLI options take precedence over config values. Calls `generate_banner(cfg=BannerConfig(...))` twice — once for light, once for dark.

## `generate qr_code`

```bash
uv run python -m scripts.cli generate qr_code \
  --config-path config.yaml \
  --output-path .github/assets/img/qr.png \
  --qr-scale 25 \
  --qr-background-path .github/assets/img/icon.svg
```

| Option | Notes |
|---|---|
| `--qr-scale` | Module size in pixels |
| `--qr-background-path` | SVG to embed behind QR modules |

## `generate word_cloud`

```bash
uv run python -m scripts.cli generate word_cloud \
  --config-path config.yaml \
  --techs-path .github/assets/topics.md \
  --output-path .github/assets/img/wordcloud_by_topic.svg \
  --prompt "Focus on ML and distributed systems"
```

| Option | Notes |
|---|---|
| `--techs-path` | Markdown file with starred-repo data (topics or languages) |
| `--output-path` | Output SVG path |
| `--prompt` | Optional LLM prompt for keyword curation |
| `--topics/--languages` | Toggle between topics or languages word cloud modes |

## `generate generative`

Generates Lorenz-attractor / neural-network / flow-field SVG art seeded from GitHub metrics.

```bash
uv run python -m scripts.cli generate generative \
  --metrics-path /tmp/metrics.json
```

Produces `generative-activity.svg`, `generative-activity-dark.svg`, `generative-community.svg`, `generative-community-dark.svg` in `.github/assets/img/`.

## `generate animated`

Generates CSS-animated SVGs seeded from commit history data.

```bash
uv run python -m scripts.cli generate animated \
  --history-path /tmp/history.json
```

Produces `animated-activity.svg`, `animated-activity-dark.svg`, `animated-community.svg`, `animated-community-dark.svg`.

## `generate skills`

Renders `skills.yaml` into shields.io badge rows and injects them into the README.

```bash
uv run python -m scripts.cli generate skills \
  --skills-path skills.yaml
```

## `generate readme`

Assembles all dynamic README sections and writes them to `README.md` via marker blocks.

```bash
uv run python -m scripts.cli generate readme \
  --config-path config.yaml
```
