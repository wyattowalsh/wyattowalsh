  <h2>Overview</h2>

  <p>Welcome to my GitHub profile page repo docs. My <i><b><code>README.md</code> profile page</b></i> is generated and maintained via a template-based system of Python scripts and GitHub Actions workflows (along with some useful extras).</p>

  <p>There are scripts (and associated files) for:</p>
  <ul style="list-style-type: none; padding-left: 0;">
    <li><i class="fas fa-image fa-fw" style="margin-right: 8px;"></i>banner image generation</li>
    <li><i class="fas fa-cogs fa-fw" style="margin-right: 8px;"></i>technologies section generation</li>
    <li><i class="fas fa-star fa-fw" style="margin-right: 8px;"></i>starred repo word cloud generation (using <code>starred</code>)</li>
    <li><i class="fas fa-qrcode fa-fw" style="margin-right: 8px;"></i>QR code generation (w/ custom background and design)</li>
    <li><i class="fas fa-file-alt fa-fw" style="margin-right: 8px;"></i><code>README</code> file generation &amp; management</li>
    <li><i class="fas fa-terminal fa-fw" style="margin-right: 8px;"></i>command line interface (CLI) for easy utilization of project scripts</li>
    <li><i class="fas fa-tasks fa-fw" style="margin-right: 8px;"></i>Makefile for efficient workflow command management</li>
    <li><i class="fab fa-github fa-fw" style="margin-right: 8px;"></i>GitHub Actions workflows for automation and continuous integration</li>
  </ul>
