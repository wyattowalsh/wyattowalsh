## 1. Project Overview

- **Name:** `wyattowalsh`
- **Type:** Personal GitHub Profile Automation & Asset Generation
- **Goal:** Dynamic, engaging GitHub profile via programmatic content generation.
- **Key Tech:** Python 3.13, `uv`, `pydantic`, `typer`, `pytest`, GitHub Actions, `Makefile`.
- **Core Features:** SVG banner, vCard QR, topic/language word clouds, WakaTime/Medium integration into `[@README.md](mdc:README.md)`.
- **Status:** Active, with ongoing enhancements focused on workflow reliability, feature expansion (e.g., `[@scripts/techs.py](mdc:scripts/techs.py)`), and consistent Python 3.13 adoption.

## 3. Core Functionality Summary

The project automates:

1.  **SVG Banner Generation:** (`[@scripts/banner.py](mdc:scripts/banner.py)`) - Complex, configurable SVG.
2.  **Artistic QR Code Generation:** (`[@scripts/qr.py](mdc:scripts/qr.py)`) - vCard PNG with SVG background.
3.  **Word Cloud Generation:** (`[@scripts/word_clouds.py](mdc:scripts/word_clouds.py)`) - From starred repo topics/languages (`[@.github/assets/topics.md](mdc:.github/assets/topics.md)`, `[@.github/assets/languages.md](mdc:.github/assets/languages.md)` which are generated by `[@.github/workflows/awesome-stars.yml](mdc:.github/workflows/awesome-stars.yml)`).
4.  **GitHub Profile Metrics SVGs:** (`[@.github/workflows/main.yml](mdc:.github/workflows/main.yml)`) - Uses `lowlighter/metrics`.
5.  **`[@README.md](mdc:README.md)` Augmentation:**
    - WakaTime stats (`[@.github/workflows/waka.yml](mdc:.github/workflows/waka.yml)`).
    - Medium blog posts (`[@.github/workflows/blog-post-workflow.yml](mdc:.github/workflows/blog-post-workflow.yml)`).
6.  **CLI for Local Generation:** (`[@scripts/cli.py](mdc:scripts/cli.py)`).
7.  **Standardized Dev Env:** `uv`, `[@Makefile](mdc:Makefile)`, linters, formatters, `pytest`.

## 4. Key Data Flows

1.  **Starred Repos -> Word Clouds:** GitHub Stars -> `[@.github/workflows/awesome-stars.yml](mdc:.github/workflows/awesome-stars.yml)` -> `[@.github/assets/topics.md](mdc:.github/assets/topics.md)` & `[@.github/assets/languages.md](mdc:.github/assets/languages.md)` -> `[@scripts/word_clouds.py](mdc:scripts/word_clouds.py)` -> PNGs.
2.  **QR Code:** Hardcoded vCard data & static SVG -> `[@scripts/qr.py](mdc:scripts/qr.py)` -> PNG.
3.  **Banner:** Pydantic Config -> `[@scripts/banner.py](mdc:scripts/banner.py)` -> SVG.
4.  **GitHub Metrics:** `[@.github/workflows/main.yml](mdc:.github/workflows/main.yml)` (lowlighter/metrics) -> GitHub API -> SVGs.
5.  **`[@README.md](mdc:README.md)` Updates:** External services (WakaTime, Medium) -> `[@.github/workflows/waka.yml](mdc:.github/workflows/waka.yml)` & `[@.github/workflows/blog-post-workflow.yml](mdc:.github/workflows/blog-post-workflow.yml)` -> `[@README.md](mdc:README.md)`.
